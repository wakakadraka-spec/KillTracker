# KillTracker

A modular, expandable Roblox universal kill feed framework.

## Goals
- Support as many Roblox combat systems as possible.
- Modular detector architecture.
- Confidence-based kill attribution.
- Draggable GUI with Kill Feed and Dev Log tabs.
- Extensible plugin system.

## Roadmap
- Phase 1: Foundation
- Phase 2: GUI
- Phase 3: Detection Engine
- Phase 4: Framework Support
- Phase 5: Advanced Heuristics
- Phase 6: Game Intelligence
